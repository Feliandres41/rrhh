# Casos de Prueba | Gestión de Colaboradores

## ID caso de prueba
CP-COLAB-005

## Título de la prueba
Guardar nuevo colaborador correctamente

## Módulo / Característica
Gestión de Colaboradores - Registro

## Descripción
Verificar que el sistema permita guardar un nuevo colaborador cuando los datos son válidos.

## Precondiciones
- Usuario autenticado.
- Estar en formulario de registro.

## Pasos para la ejecución
1. Completar todos los campos obligatorios.
2. Hacer clic en "Guardar".

## Resultado esperado
- El sistema valida la información.
- Se crea el registro en la base de datos.
- Se muestra mensaje de éxito.
- El colaborador aparece en la lista.

## Estado
No ejecutado
