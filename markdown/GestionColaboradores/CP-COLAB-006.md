# Casos de Prueba | Gestión de Colaboradores

## ID caso de prueba
CP-COLAB-006

## Título de la prueba
Guardar cambios en colaborador existente

## Módulo / Característica
Gestión de Colaboradores - Edición

## Descripción
Verificar que el sistema permita guardar los cambios realizados en un colaborador existente.

## Precondiciones
- Usuario autenticado.
- Estar en formulario de edición.

## Pasos para la ejecución
1. Modificar uno o más campos.
2. Hacer clic en "Guardar".

## Resultado esperado
- El sistema valida los datos.
- Se actualiza el registro en la base de datos.
- Se muestra mensaje de confirmación.

## Estado
No ejecutado
