# Casos de Prueba | Gestión de Colaboradores

## ID caso de prueba
CP-COLAB-004

## Título de la prueba
Carga del formulario con datos existentes para edición

## Módulo / Característica
Gestión de Colaboradores - Edición

## Descripción
Verificar que al seleccionar un colaborador existente, el sistema muestre el formulario con los datos previamente guardados.

## Precondiciones
- Usuario autenticado.
- Existir al menos un colaborador registrado.

## Pasos para la ejecución
1. Acceder al módulo "Colaboradores".
2. Seleccionar un colaborador existente.
3. Hacer clic en "Editar".

## Resultado esperado
- El sistema muestra el formulario.
- Los campos están cargados con los datos actuales del colaborador.

## Estado
No ejecutado
